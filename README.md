
# Zepto Inventory Data Analysis (SQL)

This project analyzes a cleaned Zepto inventory dataset using PostgreSQL (pgAdmin) to extract business insights.

Key Work:

Cleaned and standardized the dataset

Performed SQL analysis on pricing, discounts, stock, and categories

Calculated revenue, best-value products, and inventory metrics

Tools Used:

PostgreSQL

pgAdmin

SQL

This project demonstrates practical SQL skills for real-world inventory and pricing analysis.
